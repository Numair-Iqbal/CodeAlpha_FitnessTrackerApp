# 💪 Fitness Tracker App

A clean and intuitive Android application to track daily 
fitness activities, monitor progress, and stay motivated 
towards health goals.

## 📱 About the Project

This Fitness Tracker App was developed as part of my 
**App Development Internship at CodeAlpha**. It allows 
users to log their daily fitness activities — including 
steps, calories burned, and workout time — and visualize 
their progress through an interactive dashboard.

---

## ✨ Features

- 📊 **Daily Dashboard** — View today's fitness summary 
  at a glance
- ➕ **Log Activities** — Record exercise type, steps, 
  calories, and workout duration
- 📈 **Progress Bars** — Visual representation of daily 
  goals (Steps, Calories, Workout Time)
- ✏️ **Edit Activities** — Update logged entries anytime
- 🗑️ **Delete Activities** — Remove unwanted entries
- 📝 **Recent Activities List** — Quick overview of 
  logged history
- 💾 **Local Storage** — All data saved locally using SQLite
- 🎨 **Clean & User-Friendly UI** — Simple and motivating design

---

## 🛠️ Built With

- **Java** — Core programming language
- **Android Studio** — Development environment
- **SQLite** — Local database for storing fitness data
- **XML** — UI layout design
